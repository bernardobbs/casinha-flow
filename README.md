# Casinha Hub

> O centro de controle da sua casa.

Casinha Hub é um sistema de gestão doméstica que integra finanças familiares, compras, estoque, manutenção e planejamento da casa.

## O que é o Casinha Hub?

Mais do que um app financeiro — um **hub completo de gestão doméstica**:

- 💰 **Finanças** — receitas, despesas, orçamentos, contas a pagar
- 🛒 **Compras** — listas inteligentes que atualizam o estoque automaticamente
- 📦 **Estoque** — consumo médio, alertas de reposição, validade
- 🔧 **Manutenção** — tarefas e reparos da casa com prioridade
- 📊 **Planejamento** — revisão semanal, modo crise, situação financeira
- 🤖 **IA** — assistente doméstico com acesso aos seus dados reais

## Stack

- **Frontend:** React + TypeScript + Vite + Tailwind + TanStack Router
- **Backend:** Supabase (PostgreSQL + Auth + RLS)
- **Deploy:** Vercel
- **IA:** Anthropic Claude

## Fluxo operacional

```
Estoque baixo
    ↓
Lista de reposição automática
    ↓
Compra realizada
    ↓
Estoque atualizado
    ↓
Transação financeira criada
    ↓
Consumo médio recalculado
```
